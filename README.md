# Behind U - 香港本地 AI 廣告搜索平台

你想要的，背後都有。

## 功能特色

- 廣東話智能搜索
- PWA 支援（可添加到手機桌面）
- 全黑奢華設計
- 移動優先響應式
- Supabase 後端支援

## 技術棧

- Next.js 14 App Router
- TypeScript
- Tailwind CSS
- Supabase
- PWA

## 安裝與運行

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 構建生產版本
npm run build

# 運行生產版本
npm start
```

## 部署

項目可部署到 Vercel、Netlify 或任何支援 Next.js 的平台。
